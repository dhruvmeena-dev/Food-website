# MealBox - Food Delivery Website

A responsive food delivery landing page built with HTML and CSS.

## Features
- Responsive design with a separate mobile stylesheet (phone.css)
- Multi-section layout: Hero, Services, Clients, Contact Form, Footer
- CSS pseudo-elements for background image overlays with opacity
- Google Fonts integration
- Flexbox-based layout throughout
- Hover effects on navigation links

## Tech Stack
HTML | CSS

## Sections
- **Hero** - Welcome banner with call-to-action button
- **Services** - Food Ordering, Catering, and Bulk Ordering cards
- **Clients** - Trusted client logos section
- **Contact** - Enquiry form
- **Footer** - Copyright information

## How to Run
Download the files and open `index.html` in any browser.
